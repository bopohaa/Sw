#Назначение
Данная библиотека предназначена для последовательного выполнения задач реализованных в виде методов в помеченных специальным аттрибутом классах

#Особенности работы
* Каждый класс конкретной задачи может иметь зависимости от внешних сервисов, которые объявляются в конструкторе.
* Каждая задача может быть зависима от результата работы других задач.
* Все входящие параметры задачи должны быть результатом работы других задач, в противном случае возникнет исключение на этапе инициализации исполнителя.
* При наличии круговых зависимостей так же будет выброшено исключение на этапе инициализаци исполнителя.
* Результат работы задачи это возвращаемое значение метода задачи, если возвращаемое значение имеет тип ValueTuple то будет считаться что задача возвращает несколько значений.
* Все возвращаемые значения должны быть использованы в других задачах как входные параметры, в противном случае возникнет исключение на этапе инициализации исполнителя.
* Задача может быть выполнена синхронно или асинхронно, все зависит от результата возвращаемого после выполнения задачи (если возвращено производное значение от типа Task то эта задача будет выполнена в асинхронном режиме)

#Пример использования
```C#
[SwComponent]
class A1
{
    public void Run()
    {
        Console.WriteLine("void A1.Run()");
    }
}

...

var builder = new SwContainerBuilder();
builder.AddAssembly(typeof(Program).Assembly);
var container = builder.Build();
container.Run().Wait();
```
